Learning Journey

Master the art of bug bounty hunting through systematic learning. Understand tools deeply, learn their relationships, and progress from beginner to expert with hands-on exercises.

Your Path to Mastery

Bug bounty hunting is a journey of continuous learning. Each stage builds upon the previous one, creating a comprehensive skill set that makes you an effective security researcher.

01

Foundation Building

Master the fundamentals of web technologies, networking, and security concepts. This stage is crucial for understanding how vulnerabilities work.

Duration: 4-6 weeks
02

Tools Mastery

Learn to use essential security tools effectively. Understand not just how to use them, but when and why to use each tool in different scenarios.

Duration: 6-8 weeks
03

Hands-on Practice

Apply your knowledge in controlled environments. Practice with vulnerable applications and CTF challenges to build real-world skills.

Duration: 8-12 weeks
04

Real-World Application

Start hunting on actual bug bounty programs. Begin with VDPs and gradually move to paid programs as you build confidence and reputation.

Duration: Ongoing

Stage 2: Tools Mastery

Learn essential bug bounty tools, their relationships, and practical usage scenarios.

S
Subfinder
Subdomain Discovery

Subfinder adalah tool subdomain discovery yang menggunakan berbagai sumber passive dan active untuk menemukan subdomain yang terkait dengan target domain.

🔍 Analogi: Mesin Pencari Subdomain
Seperti Google untuk subdomain. Subfinder mencari informasi dari berbagai sumber (certificate transparency, DNS records, search engines) dan menggabungkannya menjadi satu daftar komprehensif. Mirip dengan recon-ng tapi lebih fokus pada subdomain.
🚀 Cara Penggunaan:
  • Install: go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
  • Basic usage: subfinder -d target.com -all
  • Dengan API keys: subfinder -d target.com -all -config config.yaml
  • Output ke file: subfinder -d target.com -o subdomains.txt
📈 Progressi Skill:
Basic Usage → API Integration → Automation Scripts → Custom Sources
H
httpx
HTTP Probing

httpx adalah tool untuk melakukan HTTP probing pada daftar domain/subdomain untuk mengecek mana yang aktif dan mengumpulkan informasi tentangnya.

🔍 Analogi: Metal Detector untuk Website
Seperti metal detector yang memindai area untuk menemukan logam, httpx memindai daftar domain untuk menemukan website yang aktif. Mirip dengan httprobe tapi dengan fitur yang lebih lengkap seperti technology detection dan screenshot.
🚀 Cara Penggunaan:
  • Install: go install github.com/projectdiscovery/httpx/cmd/httpx@latest
  • Basic probing: httpx -l subdomains.txt
  • Dengan status code: httpx -l subdomains.txt -status-code
  • Technology detection: httpx -l subdomains.txt -tech-detect
📈 Progressi Skill:
Basic Probing → Technology Detection → Custom Ports → Pipeline Integration
B
Burp Suite
Web Application Security

Burp Suite adalah platform komprehensif untuk web application security testing. Memiliki proxy, scanner, intruder, dan banyak tools lainnya dalam satu aplikasi.

🔍 Analogi: Swiss Army Knife untuk Web Security
Seperti Swiss Army Knife yang memiliki berbagai alat dalam satu paket, Burp Suite menggabungkan proxy, scanner, repeater, intruder, dan banyak fitur lainnya. Mirip dengan OWASP ZAP tapi dengan fitur yang lebih profesional dan komersial.
🚀 Cara Penggunaan:
  • Download dari portswigger.net dan install
  • Configure browser proxy ke 127.0.0.1:8080
  • Install CA certificate di browser
  • Mulai dengan proxy intercept OFF
  • Pelajari basic tools: Proxy, Repeater, Intruder
📈 Progressi Skill:
Basic Proxy → Manual Testing → Automated Scanning → Extensions & Macros
N
Nmap
Network Discovery

Nmap (Network Mapper) adalah tool untuk network discovery dan security auditing. Digunakan untuk port scanning, service detection, dan OS fingerprinting.

🔍 Analogi: Radar untuk Jaringan
Seperti radar yang memindai langit untuk menemukan pesawat, Nmap memindai jaringan untuk menemukan host, port, dan services yang aktif. Mirip dengan Masscan tapi dengan pendekatan yang lebih methodical dan akurat.
🚀 Cara Penggunaan:
  • Install: sudo apt-get install nmap
  • Basic scan: nmap -sV target.com
  • Full port scan: nmap -sV -p- target.com
  • OS detection: nmap -O target.com
  • Service detection: nmap -sC -sV target.com
📈 Progressi Skill:
Basic Scanning → Service Detection → NSE Scripts → Custom Scripts
S
SQLMap
SQL Injection

SQLMap adalah tool automatic SQL injection dan database takeover. Bisa mendeteksi, exploit, dan mengambil data dari database yang vulnerable.

🔍 Analogi: Master Key untuk Database
Seperti master key yang bisa membuka banyak pintu, SQLMap bisa menemukan dan memanfaatkan berbagai jenis SQL injection. Mirip dengan manual SQL injection tapi dengan automation dan intelligence yang lebih advanced.
🚀 Cara Penggunaan:
  • Install: sudo apt-get install sqlmap
  • Basic test: sqlmap -u "http://target.com/page?id=1"
  • Database enumeration: sqlmap -u "url" --dbs
  • Table enumeration: sqlmap -u "url" -D database --tables
  • Data extraction: sqlmap -u "url" -D database -T users --dump
📈 Progressi Skill:
Basic Detection → Database Enumeration → Data Extraction → WAF Bypass
N
Nuclei
Vulnerability Scanner

Nuclei adalah fast vulnerability scanner berbasis template YAML. Cocok untuk large-scale scanning dengan community-driven templates.

🔍 Analogi: Quality Control Inspector
Seperti QC inspector yang memeriksa produk dengan checklist, Nuclei memeriksa target dengan template-template yang spesifik. Mirip dengan Nikto atau Nessus tapi lebih cepat dan customizable.
🚀 Cara Penggunaan:
  • Install: go install github.com/projectdiscovery/nuclei/v2/cmd/nuclei@latest
  • Basic scan: nuclei -l targets.txt
  • Specific templates: nuclei -l targets.txt -t exposures/
  • Update templates: nuclei -ut
  • Custom output: nuclei -l targets.txt -o results.txt
📈 Progressi Skill:
Basic Scanning → Template Selection → Custom Templates → Workflows

Hands-on Exercises

1
Subdomain Discovery Challenge

Practice subdomain discovery on a target domain using multiple tools and compare results. Learn the strengths and weaknesses of each approach.

Exercise Steps:
  • Choose a target domain (use your own or a public domain)
  • Run Subfinder: subfinder -d target.com -all -o subfinder.txt
  • Run Amass: amass enum -d target.com -o amass.txt
  • Compare results: sort subfinder.txt amass.txt | uniq -d
  • Check which tool found more unique subdomains
  • Run httpx on combined results: cat *.txt | sort -u | httpx -o alive.txt
Tools Needed:
Subfinder Amass httpx Sort Uniq
2
Web Application Mapping

Learn to map a web application's attack surface using Burp Suite. Understand the application's structure and identify potential entry points.

Exercise Steps:
  • Set up Burp Suite proxy and configure browser
  • Navigate through the target application manually
  • Use Burp's Spider/Crawler to discover content
  • Analyze the site map for interesting endpoints
  • Identify all input parameters and forms
  • Create a target map with potential attack vectors
Tools Needed:
Burp Suite Browser FoxProxy
3
SQL Injection Detection

Practice detecting SQL injection vulnerabilities using both manual and automated methods. Understand the difference between various SQL injection types.

Exercise Steps:
  • Set up DVWA or similar vulnerable application
  • Manual testing with basic payloads: ' OR '1'='1
  • Use SQLMap for automatic detection: sqlmap -u "url" --dbs
  • Compare manual vs automated results
  • Try different injection types: Union, Boolean, Time-based
  • Document findings and understand the root cause
Tools Needed:
DVWA SQLMap Burp Suite Browser
4
Report Writing Practice

Practice writing clear, professional bug reports. Good communication is as important as finding vulnerabilities. Learn to document your findings effectively.

Exercise Steps:
  • Find a vulnerability in a practice application
  • Create a detailed reproduction guide
  • Include screenshots and proof of concept
  • Assess the impact and suggest remediation
  • Format the report professionally
  • Get feedback from experienced hunters
Tools Needed:
Screenshots Screen Recording Markdown Burp Suite
