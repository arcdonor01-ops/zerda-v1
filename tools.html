Reconnaissance Tools

Discover targets, enumerate subdomains, and gather intelligence about your attack surface.

S
Subfinder
Subdomain Enumeration

Fast and powerful subdomain discovery tool that uses multiple sources to find subdomains. Integrates with various APIs and passive sources for comprehensive results.

Key Features:
  • Multiple passive sources integration
  • Fast concurrent processing
  • API integration support
  • JSON output format
  • Wildcard filtering
Installation:
go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
Usage Guide GitHub
A
Amass
Network Mapping

Comprehensive tool for DNS enumeration and network mapping. Uses both active and passive techniques to discover assets and map the attack surface.

Key Features:
  • Active and passive enumeration
  • Graph visualization output
  • Integration with 50+ data sources
  • Maltego integration
  • Historical data tracking
Installation:
sudo apt-get install amass
Usage Guide GitHub
N
Nmap
Network Scanner

The network discovery and security auditing tool. Essential for port scanning, service detection, and OS fingerprinting during reconnaissance phase.

Key Features:
  • Port scanning and service detection
  • OS and version fingerprinting
  • NSE scripts for automation
  • Multiple scan techniques
  • Output in various formats
Installation:
sudo apt-get install nmap
Usage Guide Website
H
httpx
HTTP Probing

Fast and multi-purpose HTTP toolkit that runs multiple probes on target URLs. Perfect for checking which subdomains are actually live and responding.

Key Features:
  • Fast concurrent HTTP probing
  • Multiple probe types (status, title, tech)
  • Custom port support
  • JSON output format
  • CDN detection
Installation:
go install -v github.com/projectdiscovery/httpx/cmd/httpx@latest
Usage Guide GitHub

Scanning Tools

Vulnerability scanners and analysis tools to identify security issues automatically.

B
Burp Suite
Web Application Security

The industry-standard platform for web application security testing. Essential for manual testing, automated scanning, and comprehensive security assessment.

Key Features:
  • HTTP proxy for intercepting requests
  • Automated vulnerability scanner
  • Intruder for automated attacks
  • Repeater for manual testing
  • Extensive plugin ecosystem
Installation:
Download from portswigger.net
Usage Guide Website
Z
OWASP ZAP
Web Application Scanner

Free security tool for finding vulnerabilities in web applications. Great for beginners and provides both automated and manual testing capabilities.

Key Features:
  • Free and open source
  • Automated and passive scanning
  • Easy to use interface
  • API for automation
  • Great for beginners
Installation:
Download from zaproxy.org
Usage Guide Website
N
Nuclei
Vulnerability Scanner

Fast and customizable vulnerability scanner based on simple YAML templates. Perfect for large-scale scanning with community-driven templates.

Key Features:
  • Fast concurrent scanning
  • YAML-based templates
  • Community template repository
  • Customizable workflows
  • Multiple output formats
Installation:
go install -v github.com/projectdiscovery/nuclei/v2/cmd/nuclei@latest
Usage Guide GitHub
W
WhatWeb
Web Technology Detector

Next-generation web scanner that identifies technologies used by websites. Perfect for technology stack analysis and fingerprinting.

Key Features:
  • 1700+ plugin ecosystem
  • Performance tuning options
  • Multiple log formats
  • Proxy support
  • Custom plugins
Installation:
sudo apt-get install whatweb
Usage Guide GitHub

Exploitation Tools

Tools for exploiting vulnerabilities and proving their impact with proof of concepts.

S
SQLMap
SQL Injection Tool

Automatic SQL injection and database takeover tool. Essential for testing database security and extracting data from vulnerable applications.

Key Features:
  • Automatic SQL injection detection
  • Database enumeration
  • Data extraction capabilities
  • Support for multiple databases
  • Tamper scripts for WAF bypass
Installation:
sudo apt-get install sqlmap
Usage Guide GitHub
X
XSStrike
XSS Detection Tool

Advanced XSS detection and exploitation suite with intelligent payload generation and context-aware detection mechanisms.

Key Features:
  • Intelligent payload generation
  • Context-aware detection
  • WAF bypass capabilities
  • DOM XSS detection
  • Automated exploitation
Installation:
git clone https://github.com/s0md3v/XSStrike.git
Usage Guide GitHub
M
Metasploit
Penetration Testing Framework

Comprehensive penetration testing framework with thousands of exploits, payloads, and post-exploitation tools for various platforms and services.

Key Features:
  • 2000+ exploits and payloads
  • Post-exploitation modules
  • Automated exploitation
  • Multi-platform support
  • Professional reporting
Installation:
sudo apt-get install metasploit-framework
Usage Guide Website
B
BeEF
Browser Exploitation

Browser Exploitation Framework for testing browser-based vulnerabilities. Perfect for demonstrating XSS impact and browser security issues.

Key Features:
  • Browser hooking
  • Client-side attack vectors
  • Social engineering modules
  • Browser fingerprinting
  • Real-time command execution
Installation:
sudo apt-get install beef-xss
Usage Guide GitHub
